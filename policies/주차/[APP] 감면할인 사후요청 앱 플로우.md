**2️⃣ 감면/할인 버튼 노출 조건 개선 (24.7.30)**
-----------------------------------

* [**아지트 공지링크 : [T/내비앱 게시판] 이용기록 > '감면/할인' 버튼 노출 조건 개선**](https://ext.agit.in/g/300017093/wall/407886080)

|  |  |  |
| --- | --- | --- |
| **주차권 픽** | **주차패스 픽 (정기권 X)** | **주차패스 픽 (정기권 O)** |
