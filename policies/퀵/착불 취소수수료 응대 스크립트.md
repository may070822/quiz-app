| 미지급 - 미수긍(2차) | | | 제가 도움을 드릴 수 있는 방법을 확인해 보겠습니다. 잠시만 기다려 주시겠습니까?   기다려주셔서 감사합니다. 기사님의 불편사항을 고려하여 중개수수료를 제외한 취소수수료를 지급드릴 수 있도록 담당부서에 요청드리겠습니다.  다만, 착불오더의 경우 취소가 되어도 취소수수료 지급이 되지 않으니 추후 오더 수행 시에는 참고 부탁드립니다.   (TL전달) | | | | | | | | |
