[**Q1. 카카오 T 퀵을 웹페이지로도 접수할 수 있나요?**](#h_01JD8V3A7KMTFFC6X06XN375ND)
-------------------------------------------------------------------

상담유형: **퀵·배송\_고객▶앱이용/정책▶ 주문 정책▶전체**

**A1.**
-------

궁금하셨을 텐데요.

카카오 T 유저라면 카카오 T 퀵 웹 접수페이지에서 카카오 T 퀵 주문을 접수할 수 있습니다.

**[참고사항]**

1) 웹 카카오 T 퀵 접수페이지 바로가기 (링크 : quick.kakaomobility.com)

ㄴ 네이버등 포털 검색창에 카카오 T 퀵 검색 시 접수페이지 확인 가능

──────────────────────────────────────────────

[**Q2. 카카오 T 웹 접수페이지에서 쿠폰을 등록할 수 있나요?**](#h_01JD8V3A7KMTFFC6X06XN375ND)
-----------------------------------------------------------------------

상담유형: **퀵·배송\_고객▶앱이용/정책▶ 결제/요금 문의/정책▶전체**

**A2.**
-------

궁금하셨을 텐데요.

카카오 T 웹 접수페이지에서는 쿠폰을 신규로 등록할 수 없습니다.   
카카오 T앱 > 내 정보 > 쿠폰함에서 쿠폰을 먼저 등록해 주세요. T앱에서 등록한 쿠폰은 카카오 T 웹 접수페이지에서 사용하실 수 있습니다.

**[참고사항]**

1) 웹 카카오 T 퀵 접수페이지 바로가기 (링크 : quick.kakaomobility.com)

ㄴ 네이버등 포털 검색창에 카카오 T 퀵 검색 시 접수페이지 확인 가능
