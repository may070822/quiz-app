# 픽커 심사 1차 가이드 (자동차등록증)

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 픽커 심사 1차 가이드 - 자동차등록증 | | | | |

※ 처리방법

**용도가 자가용인 경우**  
1. 계정이름/이미지내 이름 동일한지 확인(다르면 반려)  
2. 용도가 자가용인 경우 차종에 승용/화물/특수 3가지 다 승인 가능  
3. 주민번호 마스킹 여부(마스킹 되어있으면 반려)  
ㄴ주민번호 앞자리는 꼭 보여야함  
4. 차대번호 입력

**용도가 영업용인 경우**  
1. 용도가 영업용인 경우 차종에 화물만 승인 가능  
2. 이름 상관 x  
3. 주민번호 마스킹 여부(마스킹 되어있으면 반려)  
ㄴ주민번호 앞자리는 꼭 보여야함  
4. 차대번호 입력 - 중간 차량번호 아,바,사,자 확인
