|  |  |  |  |  |
| 운송수단 | 필요서류 | 심사 기준 | | |
| 전동킥보드 오토바이/승용차 | 운전면허증 | ① 생년월일란에 8자리 숫자를 입력한다.(만약 성인이 아닐시 승인을 눌러도 반려처리 된다.) ② 입력한 이름과 신분증 이름이 일치하는지 확인한다. ③ 운송수단별 자격을 검증한다. (전동킥보드/오토바이 - 원동기면허 이상, 승용차 2종 보통 이상) ④ 문제 없을 시 승인 처리, 반려 사유 있을 시 반려 처리 후 사유 선택한다. ㄴ \*생년월일 식별불가로 인한 반려 진행시 생년월일란에 임의의 값 (20001111) 입력한다. | | |
| 화물차(영업용) | 운전면허증, 차량등록증 사업자등록증 | ▶ 운전면허증 확인 ( 픽커의 운전면허증이어야함 ) ① 생년월일란에 8자리 숫자를 입력한다.(만약 성인이 아닐시 승인을 눌러도 반려처리 된다.) ② 입력한 이름과 신분증 이름이 일치하는지 확인한다. ③ 운송수단(차종이 화물이고 영업용이면 라보, 다마스, 1톤이상 이외의 차종도 승인처리)별 자격을 검증한다. (적재중량 4톤 이하차량 - 2종 보통 이상) ④ 문제 없을 시 승인 처리, 반려 사유 있을 시 반려 처리 후 사유 선택한다. ㄴ \*생년월일 식별불가로 인한 반려 진행시 생년월일란에 임의의 값 (20001111) 입력한다.  ▶ 차량등록증 (픽커명의 차량이거나 , 사업자명의의 차량 확인 필요함 )  ① 입력한 차량번호와 차량등록증에 있는 차량번호가 일치하는지 확인한다. ② 용도가 영업용인지 확인한다. ③ 문제 없을 시 승인 처리, 반려 사유 있을 시 반려 처리 후 사유 선택한다 ④ 성함이 회사명으로 되어있어도 승인 가능 / 주민등록번호 뒷자리 마스킹 승인 가능 ⑤ 자동차 등록증 하나로 여러대를 등록할경우 > 반려  ▶ 사업자등록증 (법인, 개인 상관 없음) 픽커가 대표자여야함 ① 등록된 업종/업태에 운수업이 있는 지 확인한다. / 업태 화물 포함 ② 문제 없을 시 승인 처리, 반려 사유 있을 시 반려 처리 후 사유 선택한다. ③ 주민등록번호 마스킹 반려  \* 단, 공통/자격관리를 통하여 등록하는 경우 개인 사업자(업종/업태 운수업)만 등록 가능 ㄴ 본어드민에서 '공통'으로 접수되는 경우 (본어드민으로 조회 시, 심사중 서비스 '공통' 표기) | | |
