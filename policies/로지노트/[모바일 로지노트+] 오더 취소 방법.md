# [모바일 로지노트+] 오더 취소 방법

**1️⃣ 주의사항**
------------

① 하차 완료 전까지 가능

② 배차확정 전 [배차중] 상태 또는 배차확정 후 [완료] 상태 일 때,   
    콜 연동 상태가 하차완료 되기 전까지 오더취소 가능

**2️⃣****오더 취소 방법**
-------------------

**경로: '목록' 메뉴 → 대상 오더 왼쪽으로 화면 밀기 → '오더취소' 버튼 터치**
