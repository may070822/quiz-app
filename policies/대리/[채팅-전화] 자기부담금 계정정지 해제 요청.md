**① 고객 발화 시점 및 상황에 맞는 감성표현 활용 필요 (호응, 공감, 양해, 사과)**

(예시)

**② 정보확인+운행정보 확인 (특정 건 확인되는 경우, 선언급 가능)**

> (전화상담) 실례지만, 기사님 본인 성함 말씀 부탁드려요.

**③ 어드민 입금 내역 확인**

자기부담금 입금 확인하겠습니다. 잠시만 기다려주시겠습니까?

```
[대리어드민 확인]  
대리어드민 → 기사 목록 → 기사 정보 검색 → 이름 클릭 → 사고 정보  
※ 자기부담금 미입금 정지로 확인되는 경우 → 관리자 전달
```
