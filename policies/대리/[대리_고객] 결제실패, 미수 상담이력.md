# [대리_고객] 결제실패, 미수 상담이력

```
■ 주의 및 가이드 요약  
※ 고객에게 '미수' 표현 사용 지양→'OO사유로 결제 되지 않은 금액 / 결제 실패된 금액'으로 표현  
※ 이용기록에서 결제실패 금액 결제 가능

```
