| 취소수수료 | 공통 | 공통 | 다음 사유에 해당하는 경우 취소수수료가 발생할 수 있으며, 결제된 취소수수료는 환불이 불가합니다. 단, 고객 강성 항의 & 기사 귀책 주장하는 경우에는 취소수수료 환불(결제취소 또는 강제할인)이 가능합니다. | | | | | | 취소수수료 - 모범 2,000원 - 블루 2,000원 (대구/경북 지역 한정 :500원) - 벤티 4,000원 - 블랙 5,000원  ※ 취소수수료에도 동일하게 마크업(10%)가 적용 |
| 고객 단순 변심 | 택시 배차완료 1분 이후 취소하면 취소수수료가 부과됩니다. | | | | | |
| 탑승 인원 초과 | 택시 탑승 인원을 초과하여 탑승하는 경우 승차 거부될 수 있으며, 취소수수료가 부과됩니다. - 블루/모범/블랙 : 최대 탑승인원 4명 - 벤티 : 최대 탑승인원 5명 | | | | | |
| 짐 적재 기준 초과 | 화물에 의해 사고 위험 및 차량 훼손이 예상되는 물품을 차량에 적재하는 경우 기사가 승차를 거부될 수 있으며 취소수수료가 발생할 수 있습니다. - 적재 불가 : 20kg 이상의 짐, 가로/세로/높이 35cm 이상의 짐, 이삿짐 부피가 큰 가구 및 전자제품, 이동 중 짐의 파손 또는 차량 손상이 예상되는 물품 등 - 규격 외 적재 가능 : 탑승 인원수에 맞는 여행 캐리어, 골프채, 골프 가방 | | | | | |
| 노쇼 | 차량 도착 3분 이후에도 탑승하지 않는 경우, 배차가 취소될 수 있으며 취소수수료가 부과됩니다. | | | | | |
