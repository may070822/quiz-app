| 이중결제 (1) | 택시 이용완료 후 **이중결제(자동/현장)된 경우, 증빙자료를 제출하면 자동결제된 택시비를 환불**해 드립니다.   1. 증빙자료를 꼭 제출해 주세요!  - 증빙 자료(영수증 or 카드 승인문자 or 카드사 결제내역) 중 한가지를 준비해 주세요.  - 상호, 차량번호, 결제일시, 승하차시간, 결제수단 정보가 포함되어야 하며 정보 확인이 어려운 경우 도움드리기 어렵습니다.  - 증빙이 없거나 선결제 or 자동결제 승인내역 경우 도움드리기 어렵습니다.   2. 증빙자료 제출이 불가하다면 기사님께 정황을 확인해 드릴게요.  - 기사님께서 이중결제를 인정하는 경우 결제취소를 도와드립니다.  - 기사님께서 이중결제를 인정하지 않거나 증빙자료 제출이 어렵다면 도움드리기 어렵습니다.   부당하게 청구된 요금은 각 지역 관공서에 신고할 수 있습니다. | | | | | | 1) 승인번호가 없어도 가맹점명으로 현장결제로 구분이 가능한 경우 처리 가능  2) 기사 OB가 필요한 경우 ① 현장 결제수단이 현금인 경우 ② 결제금액: 1천 원 이상 차이 ③ 결제승인시간: 10분 이상 차이 ④ 기사 결제 가맹점 정보와 상이한 경우 (예외 : 스마트로) ⑤ 증빙자료가 없는 경우 ⑥ 벤티 현장결제 발생한 경우 (증빙자료는 동일하게 요청) |
| ※ 선승인을 이중결제로 오인한 문의가 있을 수 있습니다. ※ 기사 앱 자동결제 건너뛰기/결제취소 동일하게 이용 가능 ※ 이중결제 환불 요청하는 경우 증빙자료 확인 후 자동결제 취소 처리 (\*기존 국내 IN 처리와 동일) | | | | | |
| 증빙자료 ■ 카드 직접 결제 ㄴ 일시, 금액, 택시번호 or 결제 가맹점 ㄴ 승인번호(구분이 어려운 경우 확인, 어드민과 다른 번호) → 카드사 제공의 매출전표, 승인내역(문자), 지류 영수증  ■ 계좌이체 ㄴ 계좌 이체된 예금주 이름과 어드민상 기사 이름과 동일 시 인정 ㄴ 기사 이름 모두 확인되어야만 인정 (ex : 홍\*동 > 불가 / 홍길동 > 가능) ㄴ (결제금액 1천 원 이상 차이 OB 이관, 결제승인시간 기준 고려 X)  ■ 택시 차량번호 확인되는 티머니 결제 내역 ㄴ 일시, 금액, 택시번호 ㄴ 티머니 앱 결제 내역은 직접 결제 증빙 인정 | | | | | |
| 부당운임  (우회경로) | 고객님이 설정하신 출발지와 목적지가 기사에게 전송되고 기사앱에서 카카오 내비를 연결해 길안내가 이뤄지고 있습니다.  당시 도로 상황에 맞추어 최적의 이동 경로로 길안내가 되기 때문에 고객님께서 주로 이용하시는 경로와 내비게이션의 안내가 서로 다를 수 있습니다.   경로가 이탈되는 경우 **경로가 재탐색되어 우회될 수 있으며, 교통량 변화 감지로 최적의 경로로 변경**될 수도 있습니다.  만약, 탑승완료 후 부당하게 요금이 청구된 경우에는 각 지역 관공서에 부당요금으로 신고하실 수 있습니다. | | | | | |  |
| - 단순 문의 - 예상 경로를 벗어나지 않음 - 예상요금과 실결제요금의 차액 5,000원 이내 | 1차) 도로 상황에 맞추어 최적의 이동경로로 길안내가 진행되었음 설명 2차) 요금을 조정할 수 있는 행정적인 권한이 없어 지자체에 신고하실 수 있는 방법을 안내  ► 부당요금 신고 연락처 - 서울, 인천, 대전, 광주, 울산, 경기 지역 : 지역번호 + 120 - 그 외 지방 자치 단체 민원 콜센터 : 시청 등 홈페이지에 민원콜센터 번호 | | | | |  |
| - 승객 강성 항의 - 예상 경로를 벗어난 경우 - 예상요금과 실결제요금의 차액 5,000원 이상 | 택시 관리자에게 업무 이관하여 우회경로 정황 탐색  1. 예상 경로를 벗어난 경우 - 기사 OB하여 우회하게 된 정황 확인 및 부당요금 신고 접수됨 안내 - 운임 하향 조정에 동의하지 않을 경우 지자체 신고될 수 있음 안내 - 예상요금으로 운임 조정에 동의 시 재결제, 미동의 시 강제할인 처리  2. 예상 경로를 벗어나지 않은 경우 - 예상요금으로 강제할인 처리 | | | | |  |
