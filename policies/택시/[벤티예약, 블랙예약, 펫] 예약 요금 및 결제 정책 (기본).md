| 예약 요금 및 결제 정책 | | | | | | | | | | | |
| 기본 정책 | ※ ‘사전 확정 요금제' 적용 = (운행요금 \* 탄력배수) + 기타 비용 선택시간, 최적 운행 거리, 탄력 배율에 기반한 요금을 사전에 확정하여 불필요한 요금 분쟁 방지 가능 단, 승객의 경로 변경 요청, 목적지 변경 또는 경유지 추가 시 추가 요금이 발생될 수 있음  ▶ 운행요금 구분 : 시간 예약 / 일반 예약 요금제 ▶ 수요 / 공급 / 가격 수용도에 따라 탄력 배수 적용 ▶ 기타비용 : 통행료 | | | | | 선승인  (선결제) 프로세스 | 벤티  블랙 | ▶ 확정요금으로 선결제 > 예약 운행완료 > '선결제 취소 후 운임 재결제'  ㄴ 시간 예약에 경우 초과 이용이 있을시 변동된 요금으로 재결제됨 ㄴ 통행요금이 변경된 경우 실제 통행요금 반영 | | | |
| 펫 | ▶ 확정요금 선승인 확정 방식  ㄴ요금 변경 없을시 취소되지 않음 ㄴ시간 대절에 경우 초과 이용 / 통행요금 변동이 있을시 자동으로 취소 후 재결제됨 | | | |
| 일반 예약 상품 (설정한 구간에 대한 이동서비스) | 예약 접수 시 앱에서 조회 후 선결제한 금액 = 재결제 최종 금액 (펫은 선승인 확정) | | | | | 시간 예약 상품 (설정한 시간동안 자유롭게 이용하는 이동서비스) | 예약 접수 시 앱에서 조회 후 선결제한 금액에서 '거리/시간 초과하지 않은 경우' 동일한 금액으로 재결제   (펫은 선승인 확정 / 시간, 거리 초과로 요금이 변경되는 경우 자동 재결제) | | | | |
| 예약 시 앱에서 조회한 금액 = 재결제 최종 금액 - 요금 : 기본요금, 각 상품별 거리/시간요금 반영 - 예상요금 : 출발지~경유지~도착지 경로로 산정 후 확정 요금 결제 - 탄력요율 : 예약 접수 시점에 확정되며, 재산정 시에 변경되지 않음 - 예약비 : 오픈 스펙기간 무료 | | | | | - 요금 : 대여시간별 테이블 요금 - 추가요금 ㄴ 시간요금 : 대여시간 초과 시 10분 단위 부과 ㄴ 거리요금 : 제한거리 초과 시 Km 단위로 부과 - 탄력요율 : 예약 접수 시점에 확정되며, 재산정 시에 변경되지 않음 - 예약비 : 오픈 스펙기간 무료 | | | | |
| ※ 이용 정보(경로, 목적지, 경유지 등등)가 변경되어도 확정 요금으로 결제 ㄴ 변경 필요 시, 승객에게 안내 후 기사가 파트너 운영센터 통하여 재결제 요청 필요  (기존) 예약 접수 시점의 내비 경로 토대로 톨게이트 비용이 산정됨 실제 이용 시, 톨게이트 이용 정보 변경되어도 반영되지 않아 고객센터 인입 시 처리 필요 ㄴ 톨게이트 미이용으로 인입 시 제외 후 결제 ㄴ 톨게이트 추가 이용 및 톨비 금액 변경 인입 시 상호(기사↔승객)확인 후 조정 후 결제  (변경) (23년 7월 13일 15시 경~ - 일반예약 톨비 재산정 자동반영)  확정요금 선결제와 실주행 모바일미터기 통행요금 불일치 시, 실주행 톨게이트 비용으로 재산정 후 최종결제 ⓐ 최신앱 버전 기준으로 (?) 예약 안내사항과 탑승 중 (!) 요금 안내에도 안내 내용 추가 (문구 : 통행요금이 변경된 경우, 실제 발생된 통행요금으로 결제됩니다.) ⓑ 톨비 재산정 결제된 경우 이용기록에 재산정 사유 노출 | | | | | ※ 실제 이용 후 [거리, 시간, 톨비에대한 추가 요금]이 자동으로 반영하여 재결제됨  ※ 주차비는 운행요금에 미포함으로 주차비 발생 시 직접 결제 필요 | | | | |
