**[사용 방법]**

1. 카카오 T 앱 - [교통약자] 선택  
2. 코나투스 교통약자 전용 호출 웹 페이지로 자동 연결  
3. 이용약관 동의 및 사전에 등록한 정보로 본인인증 진행  
4. 인증 성공 시, 로그인 성공  
5. 웹 페이지에서 제공하는 화면에서 호출 가능 **(출-도착지가 '대전'이 아닌 경우 호출 불가)**
