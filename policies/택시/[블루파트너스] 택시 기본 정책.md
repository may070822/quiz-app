| 기사앱 가입방법 | ※ 일반으로 가입 후 블루파트너스 가맹본부를 통해 가맹 제휴 가입 문의 필요  ▶ 기사 앱 진행 flow 카카오 T 택시 기사용 앱 설치 > 필수 권한 동의 > 가입 서비스 선택(일반) > 신규 가입 > 정보 입력(연락처, 생년월일) > 약관 동의 > 휴대폰 번호 인증  > 소속 선택(개인/법인) > 프로필 사진, 영업 지역 선택 > 택시운전자격증(비치용 자격증) 첨부 > 가입신청 완료  ※ 심사 완료 후 차량정보 등록 후 출근 가능 ㄴ 차량정보 : 차량번호, 택시종류(중형, 대형)선택 → 차량모델(모델 명)선택, 유종, 하이패스 장착 여부 | | | | | | |

| 블루파트너스 기사 호출 결제수단 | 자동결제로만 호출 가능  ㄴ 단, 자동결제 호출 후 현장결제는 가능 / 자동결제 승인 후 취소가 필요 (이용료는 현장결제 불가로 자동결제로 지불) ㄴ 일반 직접결제 호출, 배회영업 운영은 직접결제 가능 | | | | | | |
| ▶ 자동결제 콜 인증방법  1. 요금 정산 제휴사 가맹하기  ㄴ 티머니, 이동의 즐거움(구 로카), 마이비, 한페이, 유페이, 스마트로 등 2. 앱 인증하기  ㄴ 메뉴 > 자동결제콜 가입하기 > 정보입력(법인은 사번 / 개인은 사업자 번호 입력) > 정보 일치 시 인증완료 및 즉시 이용 가능 3. 인증 실패  ㄴ 정보 재확인 > 정보 수정 > 재인증 진행 | | | | | | |
| 승객 호출 방식 | 고객이 카카오 T 앱에서 블루, 일반, 부스터, 바로호출로 호출 시 배차 가능 (블루파트너스 호출 시, 먼거리 백투백 배차 가능한 차량이 있는 경우 '예상 탑승 시간' 설정 가능) | | | | | | |
| 기사 콜 수락 방식 | 3초 이내 자동 수락 방식 / 즉시 수락 or 콜 멈춤 가능 콜카드 노출 시간 설정 가능 (짧게, 보통, 길게, 기본값 보통 / 기사앱 > 메뉴 > 콜카드 노출 설정)  ※ 3초 이내 콜 멈춤 가능한 정책은 승객에게 안내 불가, 자동배차임으로만 안내 | | | | | | |
| 기사의 노쇼 가능 기준 | 1. 출발지 인근 도착 (100m 이내 <- m 안내 금지) 2. 3분 초과 대기 후 신고하기로 중단 가능 | | | | | | |

| 콜 수신 상태 확인 | ▶ 기능 설명 : 콜 대기중 현재 콜 수신 상태를 확인할 수 있는 기능 (콜 수신 상태는 콜 대기 중에만 노출) - 콜 수신에 문제가 없을 경우 : '콜 수신 원활' 노출 - 콜 수신에 지연이 있을 경우 : '콜 수신 지연' 노출 다만, 단말의 GPS 설정 또는 인터넷 연결에 문제가 있을 경우, ‘지연’ 상태로 노출될 수 있음 | | | | | |
| 테스트 콜 챗봇 | ▶ 기능 설명 : 콜 수신이 가능한 상태인지 콜 카드 수신 테스트 기능을 챗봇을 통해 이용 가능  ▶ 이용 가능 횟수 1일 2회 ▶ 이용 방법 콜 대기 화면 좌측 상단의 '콜 수신 원활 또는 지연' 영역을 선택 시  현 위치 정보와 테스트 콜을 받아볼 수 있는 팝업 제공 | | | | | |
| 시계외 지역 콜 표기 | ▶ 목적지가 기사의 사업구역 외 지역인 경우 공동사업구역을 포함하여 콜카드 중앙에 [시계외 지역] 표기 ㄴ 어드민에 등록되어 있는 운행 지역 기준  ▶ 시계외로 운행을 하기 어려울 경우 참고하실 수 있도록 태그 제공 (블루파트너스는 목적지를 확인할 수 없기 때문)  ※ 시계외 콜 표기는 시외할증 부과여부에 대한 표기가 아님 ※ 목적지가 공동사업구역이라도 기사가 등록된 사업구역 외라면 표기 ㄴ e.g 서울이 사업구역인 기사님에게 목적지 인천공항(공동사업구역) 콜 배분 시, [시계외 지역] 표기 | | | | | |
| 운행완료 콜멈춤 예약 | 운행완료 후 '휴식, 이동, 식사 등등' 콜 운행을 멈출 예정인 경우 미리 콜멈춤 설정 가능 | | | | | |
