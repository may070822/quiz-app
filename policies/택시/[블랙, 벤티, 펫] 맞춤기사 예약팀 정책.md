| 서비스 소개 / 목적 | 이용 경험이 좋았던 기사님을 지정하여 호출을 요청할 수 있는 기능 공급자는 개인 영업을 통한 수요를 확보하고 단골 유저를 확보하기 위해 자발적으로 더 나은 서비스 제공 고객은 이동 목적이 명확할 수록 차종, 서비스 퀄리티 등 관련 목적에 부합한 기사와 매칭을 희망하여 원하는 이동 계획 수립에 용이할 것으로 기대 | | | | | | |
| 서비스 지역 | ▶ 예약 서비스 지역 전체 ㄴ 벤티 : 서울, 인천, 경기 전체(부천, 시흥, 고양, 용인, 안산, 군포, 안양, 의왕, 과천, 성남, 의정부, 평택, 김포, 광명), 광주광역시,전남, 충남, 충북, 제주, 대전 ㄴ 블랙 : 서울, 인천, 경기, 부산, 경남(창원), 충남 ㄴ 펫 : 서울, 인천, 경기(수도권) | | | | | | |
| 호출정책 / 요금체계 | - 일반적인 예약과 동일한 정책 및 요금 / 취소수수료 정책 동일 - 예약비 일정 기간 무료 운영 | | | | | | |
| 요청사항 | -예약단계에서 유저는 요청사항을 자유롭게 기재 가능, 기사는 요청사항 확인 후 수락 (30자 이내 입력 가능 / ex. 캐리어1개가 있어요.) -접수 후 요청사항 변경은 불가 (필요 시 기사와 별도 협의하도록 안내) ㄴ 어드민에서 '변경' 버튼 활용 금지 | | | | | | |
| 콜카드 | - 고객이 선택한 기사 or 예약팀 소속 기사에게만 콜카드가 보여짐 - 기사 선택인 경우, 기사앱에서 별도 뱃지[단독]로 구분되어 표기됨 - 맞춤콜은 항상 출발시간 빠른순으로 노출 (필터/정렬 기능 제공 안함, 기본 예약콜 리스트는 제공)  - 예약접수 상태의 맞춤콜 노출, 최대 10개까지 노출되며 10개 초과 시 비 노출  - 예약팀으로 선택된 콜카드는 다른 기사가 수락하는 경우 더이상 노출하지 않음 - 맞춤콜이 없는 경우에는 없음 안내 문구 노출 | | | | | | |
| 맞춤기사 선택 | 고객은 원하는 기사 or 예약팀을 선택할 수 있음 | | | | | | |
| 기사 선택 방법 | 1. 최근 1년 간 운행한 기사 중 선택 (최대 3명 노출 / 호출 포함 최근 호출순) | | | | | |
| 2. 기사 명함의 QR코드로 검색하여 선택 (벤티/블랙/펫-명함은 온라인몰에서 크루가 직접 구매 가능) | | | | | |
| 3. 차량번호로 검색하여 선택 | | | | | |
| ※ 기사는 1명만 선택할 수 있으며, 기사 선택 시 예약팀 선택 불가 | | | | | |
| 추천기사 선택 (예약팀) | 카카오 T 맞춤기사 선택 > 추천기사 예약팀 중 선택 | | | | | |
| 예약팀은 1개만 선택할 수 있으며, 예약팀 시 기사 선택 불가 | | | | | |
| 예약팀 정책 | - 예약팀은 기사님들로 구성된 팀을 뜻 함 - 서비스 품질, 차량특성, 특정이용 목적 등 예약을 이용하는 승객이 선호하는 주요 특성을 고려하여 생성  - 기사앱 > 메뉴 > 내정보 수정 > 화면 하단의 '예약팀' 소속 정보 노출 | | | | | | |
| - 예약팀에는 다양한 사업구역 기사들이 소속될 수 있음 - 기사는 여러 예약팀에 중복하여 소속될 수 있음 (소속 개수 제한 없음) - 카카오 T 앱에서 노출되는 공개 예약팀은 최대 10개까지 리스트 노출 됨 - 카카오 T 앱에서 노출되지 않는 비공개 예약팀은 QR 코드로 검색하여 선택 | | | | | | |
| - 기사가 탈퇴/휴면상태로 변경되는 경우 해당 예약팀에서 자동 제외됨  - 최소 그룹원수는 1명이며, 그룹원수가 0명이 된 예약팀은 자동 삭제됨  [벤티](기준: 매월 1일 기준 직전 1개월/24.02.26 시행) - (신규) 공항전문 예약 콜 수락 이력이 없는 경우에 예약팀에서 제외 (고객 취소 제외) - 예약취소 3건 이상, 예약미운행 1건 이상 2가지 조건에 하나라도 해당하는 경우에는 다음달 공항전문 예약팀에서 제외 (고객 취소 제외) - 캐리어 서비스, 안전운전, 정시도착을 수행하지 않아 민원이 접수되는 경우 크루님과 사실관계 확인 후 제외 - (신규) 비정상적인 콜 수락 및 운행 이력이 확인되는 경우에는 모든 예약팀에 영구적으로 제외 | | | | | | |
| 예약팀 탈퇴 후 재 가입은 2개월 후에 가능하며 교육을 재수료 필요 (본인이 제외요청한 경우 포함) ► 탈퇴 이력 확인 방법 : 택시 어드민 > 기사 그룹 > 변경 이력 > 기사 id 로 검색 | | | | | | |
