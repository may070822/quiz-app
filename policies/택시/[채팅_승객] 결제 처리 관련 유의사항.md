# [채팅_승객] 결제 처리 관련 유의사항

| 어드민 처리 | 어드민 처리 구분/방법 | - 빌링 Admin : 요금, 플랫폼취소수수료, 포인트 - 택시 Admin : 기사 차단 및 내부신고 / 상태값 변경  ※ [운행 중 → 운행완료] 처리 건 ㄴ 요금, 플랫폼수수료, 포인트, 상태값 변경은 택시 Admin 에서 처리 | | | | 빌링어드민 처리 시 사유입력 | 문의일시, 처리 사유, SMS발송 또는 승객동의여부 ㄴ ex ) 2022-05-24 / 이중결제 취소 / 기사 SMS ㄴ ex ) 2020-11-03 / 결제카드변경 | | | | |
| 해외발급카드 | - 선결제 취소 지연 발생할 수 있음  - 카드/매입사 정채긍로 3일 이내 재결제 시 부정 사용으로 의심하여 재결제 실패 (요금 조정 필요 시, 3일 후 처리 필요) | | | | 빌링어드민 '미수철회' 기능 | 자동결제 콜 호출 후 카드결제 실패로 미수금 발생, 이후 기사에게 직접결제하여 이중결제 발생 └이중결제 증빙자료 확인 후 '미수철회' 진행 | | | | |
| 이중결제 증빙 이미지 전송 | 채팅 상담 CEMS를 통해 이미지로 수신 | | | | 블랙\_빌링어드민 처리시 주의사항 | ▶ 블랙 이중결제 취소 └빌링어드민 - 자동결제→현장결제 변경처리  └결제 금액 변경 X \_ 현장결제 금액이 자동결제 금액과 다른 경우 현장결제 금액을 반영하여 재결제 | | | | |
| 내비 경로 | 기사 탈퇴 시, [콜 상세 > 지도 > '픽업 / 주행'] 선택 불가  요금 분쟁으로 경로 확인이 필요하나 기사 탈퇴로 확인 불가 시, 아지트 CS 작성 → TL확인 후 운영매니저 이관  ex.) 미탑승, 경로 우회, 모바일 앱 미터기 요금 불일치 문의 인입하였으나 기사 탈퇴로 내비 및 주행 경로 확인 불가한 상황 | | | | ▶ 블랙 취소수수료 미수회수 └결제카드 확인하여 '자동결제 미수회수'로 진행 ('현장결제 미수회수' 진행X) └'자동결제 미수회수' 진행 불가한 경우 모빌리티 계좌로 입금 안내  (아지트 작성 가이드 참고) | | | | |
| 빌링어드민 1년 이상의 거래 건 결제 취소 불가 | ▶ 거래 1년 경과 건은 조회만 가능  ▶ 결제 취소 필요 시, 현금 환불 필요 ㄴ '신분증, 통장 사본' 개인정보 수집 동의 및 정보 확인 후 CS 아지트 작성  ※ 본인 명의가 아닌 정보로 요청 시, 정보가 잘못된 경우 도움드리기 어려울 수 있음 | | | | |
| 재인입 | 처리가 언제되는지 문의 시 | ▶ 카카오 T 고객센터로 요청하였다는 경우 → MCS, CEMS 상담이력 조회 ㄴ 아지트 / 지라 이관된 경우 작성된 내용 확인 및 처리 진행도 확인 안내 ㄴ 답변이 작성된 건인 경우 상담 중 확인 후 안내 진행  ▶ 그외 고객이 게시판으로 문의 또는 유선 접수했다고 하는 경우 ㄴ MCRM 상담이력 조회 → 순차적으로 진행됨 안내 후 종결 | | | | | | | | | |
| 아지트 작성 | 결제 및 운행 분쟁이 생기는 경우 가이드에 따라 각 CS 아지트의 글 양식으로 작성 필요 문의자의 주장에 대해서 기사(크루)에게 현장 상황과 사실 확인이 필요한 경우 OB를 진행하여 함께 전달  ※ 아지트 작성하여 전달할 때에는 문의 일시와 OB 진행 일시를 함께 작성 ※ TID / 콜 ID는 URL 링크로 작성 | | | | | | [사실 확인 및 현장 상황에 대한 확인이 필요한 문의] - 경유지 추가 분쟁  - 운행 중 발생하는 분쟁  - 결제 관련 분쟁  - 이 외 사실 확인 및 현장 확인이 필요한 문의 | | | | |
| OB 이관 시, 필수 작성 내용 | ※ 문의 내용, 이관 요청 사항'에 각 문의 유형에 맞는 필수 내용 기재 ------------------------------------------------------------------------------  ■ 문의 내용 :  (앱 오류일 경우 - 휴대폰 모델명 / OS 정보/ 통신사/ 앱 버전 기재)   ■ 이관 요청 사항 : | | | | | | | | | | |
| 이중결제 | ㅁ 이중결제 이관한 사유  (시간 00분 차이 or 금액 0000원 차이 or 결제 가맹점 상이 등....)  ㄴ 증빙자료 미비인 경우 왜 미비인지 사유 상세 작성  ㅁ 현장결제수단(카드or현금) :  ㅁ 현장결제 : [카드] 금액, 일시, 승인 가맹점 / [현금] 금액  ㅁ 자동결제 : 금액, 일시  (\*승인 가맹점 : 어드민과 증빙 가맹점 상이한 경우 기재) | | | | | OB 분실물 확인 요청 시 | ㅁ분실 물품 상세 정보 ㄴ 물품 특징까지 전달되어야 확인에 용이함 (ex. 검은색 지갑 / 죠르디 모양 에어팟 / 분홍색 백팩 등..) | | | |
| 미탑승 | ㅁ 미탑승인데 즉시 처리 못하고 이관한 사유 ㅁ 고객이 기사와 통화한 내용 / 탑승하지 못한 사유 ㅁ (필요시) 대리 호출 여부 / 목적지 변경 여부 / 경유지 추가 여부 탐색 | | | | | OB 연락처 확인 요청 시 | ㅁ연락처 확인 요청하는 사유 기재 ㄴ 이관 기준 부합 문의인지 내용 및 전산 파악  (분실물/사고/운임/예약) | | | |
| 예약 취소수수료 | ㅁ 기사 귀책으로 판단한 사유 ㅁ 당사 (카카오 모빌리티) 앱 문제로 인해 발생하였다고 판단한 사유 | | | | | 하차 후 운행중 | [고객> 기사 연락하였으나 연락두절 / 고객센터 지속 처리 요청]  ㅁ 현장 결제 여부  ㅁ 현장 결제 후 하차한 경우 결제수단과 금액 | | | |
