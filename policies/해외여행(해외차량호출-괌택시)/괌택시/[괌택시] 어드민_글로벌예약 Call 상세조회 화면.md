|  | **① 콜 ID**  **② 파트너 콜 ID**  - Copy text 기능 포함  **③ T 유저 ID** : 고객 유저 ID  **④ 서비스지역** : GUAM  **⑤ 상태값** : 차량 운행 상태 (사용 X) |
