# 특이민원 제보

#### 기본 프로세스 : D+1영업일 민원팀 검수 → 추가대응 필요 건 검토하여 이슈 우려 건 자체 이관

D+1영업일 이전 빠른 제보 필요 / 민원팀 이관 검토 필요 건 : 태그 등록 + 관리자가 각 서비스 별 아지트 글양식을 통해 글 작성
